# Day 65 — Scientific Calculator

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A fully functional scientific calculator with history,
keyboard support and engineering functions.

## How to view
Open `index.html` in your browser.

## Features
- Basic and scientific operations
- Trigonometric functions (sin, cos, tan)
- Logarithm, power, factorial, constants
- Degree and Radian mode
- Calculation history
- Full keyboard support
